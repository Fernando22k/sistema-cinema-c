# 🎬 Sistema de Gerenciamento de Cinema em C

## 📖 Descrição
Este projeto foi desenvolvido em linguagem C como atividade acadêmica, com o objetivo de aplicar conceitos fundamentais de programação, como estruturas de dados, manipulação de memória e organização de sistemas.

---

## 🎯 Objetivo
Simular o funcionamento de um sistema de cinema, permitindo o gerenciamento de clientes, ingressos e fluxo de atendimento.

---

## 📚 Contexto Acadêmico
Projeto desenvolvido durante as aulas da disciplina de Programação em C, com foco na aplicação prática dos seguintes conceitos:

- Estruturas de dados (filas e pilhas)
- Lógica de programação
- Organização de código
- Manipulação de dados em memória

---

## ⚙️ Funcionalidades

- Cadastro de clientes
- Organização de atendimento em fila
- Controle de ingressos
- Simulação de atendimento
- Estrutura de fila circular
- Operações de inserção e remoção

---

## 🛠️ Tecnologias Utilizadas

- Linguagem C
- Compilador GCC
